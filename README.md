# PID – Plataforma Inteligente de Destinos

Este repositorio forma parte del proyecto **PID (Plataforma Inteligente de Destinos)**, una iniciativa orientada al diseño, desarrollo y evaluación de soluciones basadas en datos para mejorar la gestión, análisis y toma de decisiones en destinos turísticos inteligentes.

El proyecto se desarrolla en el marco de la colaboración entre la **Universitat de les Illes Balears (UIB)**, a través del grupo **MoDAE**, y entidades públicas vinculadas a la gestión turística del territorio.

---

## 🎯 Objetivos del proyecto

Los principales objetivos de PID son:

- Integrar datos heterogéneos procedentes de distintas fuentes (turísticas, territoriales, sociales, etc.).
- Desarrollar modelos analíticos y predictivos para la toma de decisiones.
- Aplicar técnicas de ciencia de datos, estadística y aprendizaje automático.
- Facilitar la visualización y explotación del conocimiento generado.
- Apoyar la planificación estratégica de destinos turísticos inteligentes.
