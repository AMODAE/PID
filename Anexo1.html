Anexo X. Ejemplo simplificado de ontología y grafo de conocimiento para MO-4

Representación didáctica de un activo turístico mediante R

Autor/a

Equipo UIB - IAIB / MoDAE / GREI

1 Objetivo del anexo

Este anexo muestra, de forma simplificada, cómo puede representarse un activo turístico del módulo MO-4 como un pequeño grafo de conocimiento.

El objetivo no es sustituir la arquitectura real de la PID, ni reproducir la Ontología del Turismo de SEGITTUR de forma completa. El objetivo es mostrar de manera comprensible cómo una ontología define clases y relaciones, y cómo un grafo permite representar activos concretos siguiendo esa estructura.

La idea central es la siguiente:

La ontología define el significado.
El grafo conecta los datos reales siguiendo ese significado.

En este ejemplo se usa el activo turístico Playa de Muro. Podría sustituirse por cualquier otro activo del catálogo MO-4: un museo, una oficina de turismo, una ruta, un evento, un alojamiento o un servicio público.
